description: Configure Claude Needs You notification settings
disable-model-invocation: true
---

To configure notification settings, run this in your terminal:

```
~/.config/claude-needs-you/settings
```

This opens an interactive menu with arrow-key navigation to change mode, sound, delay, and debug settings. No tokens used.

(The launcher is created automatically the first time a notification fires. If it doesn't exist yet, trigger a test notification or run: `node ~/.claude/plugins/cache/jovonbuilds/claude-needs-you/*/scripts/settings.mjs`)
